SFP Planner for Ruby
====================
- Author: Herry (herry13@gmail.com)
- Version: 0.0.1
- License: [BSD License](https://github.com/herry13/sfp-ruby/blob/master/LICENSE)

A gem that provides a Ruby interface to an SFP agent.


To install
----------

	$ gem install sfpagent


Requirements
------------
- Ruby (>= 1.8.7)
- Rubygems
	- sfp (>= 0.3.0)
	- antlr3
	- json
